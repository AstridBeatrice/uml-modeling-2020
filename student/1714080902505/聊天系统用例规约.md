# 聊天系统用例规约

## 表1：添加好友用例规约

| 用例编号 | UC01                   |
| -------- | ---------------------- |
| 用例名称 | 添加好友               |
| 前置条件 | 用户已经绑定手机号     |
| 后置条件 | 发起聊天               |
| 基本步骤 | 1.用户输入手机号码     |
|          | 2.系统搜索手机号码     |
|          | 3.用户发起添加请求     |
|          | 4.对方同意请求         |
| 扩展流程 | 1.系统搜索不到手机号码 |
|          | 2.对方不同意请求       |
