## 表1:添加好友用例规约

| 用例编号 | UC01                                |
| -------- | ----------------------------------- |
| 用例名称 | 添加好友                            |
| 前置条件 | 用户绑定手机号码                    |
| 后置条件 | 用户发起聊天                        |
| 基本步骤 | 1.用户输入手机号码                  |
|          | 2.用户点击添加按钮                  |
|          | 3系统推送添加请求                   |
|          | 4.对方同意请求                      |
|          | 6系统将双方分别添加到对方的好友列表 |
| 拓展流程 | 系统检查输入手机号码为空，提示失败  |
|          |                                     |
|          |                                     |

| 用例编号 | UC02                               |
| -------- | ---------------------------------- |
| 用例名称 | 聊天                               |
| 前置条件 | 对方在该用户的好友列表             |
| 后置条件 | 无                                 |
| 基本步骤 | 1.用户输入信息                     |
|          | 2.用户点击发送按钮                 |
|          | 3系统判断双方是否互为好友          |
|          | 4.系统保存信息                     |
|          | 5.系统传输消息给消息接收者         |
|          | 6系统通知对方                      |
| 拓展流程 | 系统检查双方不是互为好友，提示失败 |
